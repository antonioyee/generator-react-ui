# generator-react-ui
 Yeoman generator for React with Material-UI and Redux


## Prerequisites ##
1. Install [yeoman](http://yeoman.io/): `npm install -g yo`
2. Install this: `npm install -g generator-react-ui`


## Use `generator-react-ui` ##
`yo react-ui`
